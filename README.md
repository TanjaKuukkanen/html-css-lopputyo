# 🌍 HTML & CSS lopputyö

Tämä projekti on ensimmäisen vuoden opintojen aikana toteutettu staattinen verkkosivusto, jossa harjoittelin HTML- ja CSS-perusteita sekä monisivuisen sivuston rakentamista.

👉 [Avaa sivusto](https://tanjakuukkanen.github.io/html-css-lopputyo/)

## 🧠 Projektista

Sivusto on toteutettu useana HTML-sivuna, jotka on yhdistetty toisiinsa navigaation avulla.
Ulkoasu on rakennettu erillisellä CSS-tiedostolla ja Bootstrapia on hyödynnetty osassa komponentteja.

Projektin tavoitteena oli oppia:

HTML perusrakenne
CSS-tyylittely
monisivuisen sivuston rakentaminen
perus responsiivisuus

## 🛠️ Käytetyt teknologiat
HTML5
CSS3
Bootstrap

## 🚀 Mitä opin

Projektin aikana opin erityisesti:

miten sivut linkitetään toisiinsa
miten ulkoasu rakennetaan CSS avulla
miten sivurakenne vaikuttaa käytettävyyteen
miten Bootstrapia voidaan hyödyntää layoutissa

## 💡 Jatkokehitys

Jos jatkaisin projektia, kehittäisin sitä esimerkiksi:

parantamalla responsiivisuutta mobiililaitteille
lisäämällä interaktiivisuutta JavaScriptillä
modernisoimalla ulkoasua
